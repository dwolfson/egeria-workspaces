Dr.Egeria Sample Files

This directory contains sample files for Dr.Egeria. These files can be used as templates or references when creating your own Dr.Egeria markdown files.

Directory Contents

How to Use

To use these sample files:

  1. Review the files to understand the structure and syntax of Dr.Egeria markdown
  2. Create your own markdown files based on these templates
  3. Place your files in the Dr.Egeria Inbox
  4. Check the Dr.Egeria Outbox for results
